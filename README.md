# Disbase
Disbase is a Discord bot that use Discord chat as the place you upload images, videos or files.

# Feature
- [ ] Image/Videos type re-uploader
- [ ] Infinity storage system
- [ ] Short link creator
- [ ] Password Manager
- [ ] Reminder

# Ideal Discord server template
```
- [ Images ]
| - upload channel
| - output channel
| - statics
```
